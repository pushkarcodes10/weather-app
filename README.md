# Weather App

A simple weather application that provides real-time weather information for any city using a public weather API. This project focuses on core functionality and clean UI, but it is currently not responsive and is best viewed on desktop screens.

---

## Features

- Search weather by city name  
- Displays temperature and weather conditions  
- Shows humidity and wind speed  
- Fetches real-time data from a weather API  
- Simple and clean user interface  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- Open Weather API

---

## Setup

1. Create a `config.js` file in the root folder
2. Add the following:

You can get your open weather api key from https://home.openweathermap.org/api_keys

``` Java-Script
const API_KEY = "your_api_key_here";
```
